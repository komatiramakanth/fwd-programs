📖 What is Clickjacking?

Clickjacking (UI Redress Attack) tricks users into clicking something different from what they perceive. An invisible iframe overlays a fake UI element — the user thinks they click a gift button but actually triggers a hidden action on a legitimate site.

🛡️ Security Headers That Prevent It

HeaderValueWhat it does
X-Frame-Options DENY Blocks page from loading inside any iframe
Content-Security-Policy frame-ancestors 'none' Modern replacement for X-Frame-Options
X-XSS-Protection 1; mode=block Browser blocks detected XSS attacks
X-Content-Type-Options nosniff Stops browser guessing MIME types
Referrer-Policy no-referrer Hides where the request came from
